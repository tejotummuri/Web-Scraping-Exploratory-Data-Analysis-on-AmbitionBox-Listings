Web Scraping & Exploratory Data Analysis on AmbitionBox Listings

This project involves collecting real-time company data from AmbitionBox using web scraping techniques and performing Exploratory Data Analysis (EDA) to extract meaningful insights about company performance, ratings, and user reviews.

🚀 Project Objectives
Scrape company data such as ratings, reviews, salaries, jobs, and benefits
Clean and preprocess raw data for analysis
Perform EDA to identify patterns, trends, and relationships
Analyze company ratings and review distributions
🛠️ Technologies Used
Python
Web Scraping (BeautifulSoup / Requests / Selenium)
Pandas & NumPy
Matplotlib / Seaborn
📂 Dataset Features
Company Name
Industry
Branches
Location
Ratings
Reviews
Salaries
Jobs
Interviews
Benefits
Awards
🔍 Key Insights
Most companies have ratings between 3.5 and 4.0, with a median around 3.9
IT Services & Consulting sector has the highest number of companies
Salary-related reviews dominate, indicating strong user interest in compensation
Strong positive correlation between Reviews, Salaries, Benefits, and Interviews
Only a small number of companies achieve very high ratings
📈 Analysis Performed
Univariate Analysis (distribution of ratings, reviews, industries)
Bivariate Analysis (relationships between features)
Correlation Analysis (heatmaps for feature relationships)
🧹 Data Cleaning
Removed irrelevant columns
Handled missing values
Structured the dataset for analysis
📌 Conclusion

This project highlights how web scraping and data analysis can be combined to derive valuable insights from real-world data. It demonstrates skills in data collection, preprocessing, and visualization.
